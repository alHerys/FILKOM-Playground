Home Structure Computer Vision

AI & Data Science • Created on 09 July 2020
Created By: Daniel Smite Manalu
Description:
An AI platform that uses computer vision to analyze photos of a room, allowing users to visualize interior design changes.
The platform is powered by a set of cutting-edge Computer Vision and Machine Learning technologies:
  • Semantic Segmentation: Algorithms that label every pixel in an image to differentiate between walls, floors, furniture, and other elements.
  • Object Detection: Models trained to recognize and locate specific objects like sofas, tables, appliances, and even people within a room.
Our AI then works to:
  1. Identify Core Structures: Differentiate between permanent elements (walls, floors, ceilings, doors, windows) and dynamic ones (furniture, decor, people).
  2. Build a 3D Model: Reconstruct the roomâ€™s layout into a dimensionally accurate 3D model.
  3. Apply Contextual Labels: Recognize the room type (kitchen, bedroom, living room) and key objects within it.
